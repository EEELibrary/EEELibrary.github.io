Spacing and color

line-height & letter-spacing

On block level elements,
the line-height property specifies the minimum
height of line boxes within the element.

word-spacing & color

The letter-spacing CSS property specifies spacing behavior between text characters. In typography, letter spacing is known as kerning. The word-spacing CSS property specifies the spacing behavior between words. The color property sets the foreground color of an element's text content, and its decorations. It doesn't affect any other characteristic of the element; it should really be called text-color and would have been named so, save for historical reasons. Note that the color value must be a uniform color, which may include a transparency value from CSS3 onwards.
